报错原因:

实体类的类名与数据库表的表名不一致导致报错

解决方案:

将实体类的类名改为数据库表名让其一致或者使用注解@TableName在实体类上定义为数据库表名

范例:

- 实体类

```java
@SuppressWarnings("all")
@Data
//设置数据库表的名称,解决实体类与数据库表名不一致情况
@TableName("plus")
public class Plusa {
    private Long id;
    private String name;
    @TableField(value="password")
    private String pwd;
    private Integer age;
    private String tel;
}
```
